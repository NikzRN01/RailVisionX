# RailVisionX

Project skeleton for a multi-camera real-time/offline pipeline.

## Layout
- `apps/`: runnable entrypoints
- `src/`: library code
- `configs/`: configuration files
- `data/`: datasets and samples
- `models/`: checkpoints/ONNX/TensorRT artifacts
- `outputs/`: logs, frames, analytics outputs
- `scripts/`: helper shell scripts
